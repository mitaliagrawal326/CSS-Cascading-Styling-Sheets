There are five types of CSS Selectors:
  1. Universal Selector
  2. Id Selector
  3. Class Selector
  4. Grouping Selector
  5. Element Selector

(*) Universal Selector is applied on background color.

(#) The CSS rule applied to the HTML element with id="para".

(.) HTML elements with class="center" will be bold and italic.

Grouping Selector- We take all the elements to apply the same properties in one selector.
